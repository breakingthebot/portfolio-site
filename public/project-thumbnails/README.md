# Project thumbnails

Drop an image here to give a specific build a real thumbnail on the Projects page, instead of the placeholder icon.

**Naming:** `{build_number}.jpg` — for example, `25.jpg` for build #25 (FlavorFind Recipe Finder). The build number is the one shown at [breakingthebot.github.io/286-builds](https://breakingthebot.github.io/286-builds/).

That's it — no code changes needed. Once the file is here and pushed, the site detects it automatically on the next deploy and swaps the placeholder for your real image.
